# Client

Frontend for Web3 OAuth. Was built using React + Vite + Typescript.

## Key Pages

1.  Landing page
2.  Connect wallet
3.  Loader
4.  Card creation
5.  PFP selection
6.  Card management
7.  Error page

## Installation & Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

    npm install

To start server:

    npm run dev

To visit app:

    http://localhost:5173
